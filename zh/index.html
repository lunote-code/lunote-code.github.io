    写作、双链、知识图谱

    Lunote 将流畅的可视化编辑与内建的 [[双链]]、反向链接、知识图谱融为一体。 打开任意 .md 文件夹即可上手——无需导入、无需账号、可完全离线。

    下载 Lunote 查看源码 阅读博客

    核心能力，开箱即用

    写作、双链、导出与跨平台支持全部内建——上手不必先折腾一圈插件。

    写作

    Typora 式编辑

    可视化与源码模式、斜杠菜单、表格、公式、Mermaid、标注框与专注模式。

    双链

    Obsidian 式知识网络

    [[双链]]、反向链接、局部图谱、标签与 YAML Frontmatter，全部写在纯文本文件里。

    导出

    需要时再导出

    支持导出 PDF、Word（DOCX）、HTML、PNG，便于分享或归档。

    平台

    三大桌面平台

    macOS、Windows、Linux 原生客户端，界面支持 11 种语言。

    用主题定制你的界面

    六套内置主题、主题文件夹自定义 CSS 与 JSON 样式变量，以及可选的社区主题目录——写作与双链不依赖插件。

    内置

    浅色 / 深色主题

    GitHub、IDEA、Dim 各提供浅色与深色版本。在主题菜单或偏好设置中切换。

    主题文件夹

    你的样式，你做主

    将 CSS、代码片段与 JSON 样式变量放入 ~/.luna/theme/，重新扫描即可生效。

    社区

    主题插件包

    在偏好设置 → 插件中浏览、安装社区主题包，一键换上现成风格。
